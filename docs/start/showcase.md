### [PR Review → Telegram Feedback](https://x.com/i/status/2010878524543131691)
**@bangnokia** • `review` `github` `telegram`

OpenCode finishes the change → opens a PR → OpenSoul reviews the diff and replies in Telegram with “minor suggestions” plus a clear merge verdict (including critical fixes to apply first).

### [Wine Cellar Skill in Minutes](https://x.com/i/status/2010916352454791216)
**@prades_maxime** • `skills` `local` `csv`

Asked “Robby” (@opensoul) for a local wine cellar skill. It requests a sample CSV export + where to store it, then builds/tests the skill fast (962 bottles in the example).
